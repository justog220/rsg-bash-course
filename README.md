Este repositorio corresponde a la edición 2026 del curso introductorio a Bash del RSG Argentina. Reúne el material teórico, los ejemplos y los archivos de práctica para seguir el recorrido completo del curso.

## Cómo usar este repo

- El recorrido está pensado en orden, de principio a fin.
- Cada módulo es autocontenido y tiene sus propios datos.
- Los ejemplos están pensados para ejecutarse desde la terminal dentro de este proyecto.
